Site Ana Ruiz – Mode d'emploi

1. Placez vos images d’origine dans le dossier /img/ (garde les noms exacts, sauf "signature" et "photo-ana" exclus du diaporama).
2. Placez le fichier audio du diaporama dans /audio/diapo.mp3 (garde ce nom exact).
3. Ouvre index.html dans ton navigateur pour tester le site.
4. Le diaporama défile automatiquement toutes les images (sauf signature/photo-ana) synchronisées avec la musique.
5. Les galeries sont 100% fidèles à la structure d’origine.
6. Pour changer un texte, modifiez simplement le fichier HTML concerné.

Tu peux mettre ce site en ligne sur n’importe quel hébergement statique.
